# Licenciatura en Estadística – Proyectos académicos

Repositorio con trabajos prácticos y finales realizados durante la
Licenciatura en Estadística (Universidad de la República).

Incluye proyectos de:
- Modelos Lineales
- Modelos Multinivel
- Inferencia Bayesiana y MCMC
- Meta-análisis
- Demografía y proyecciones poblacionales
- Programación y métodos computacionales

Cada carpeta contiene el informe del proyecto y su README específico.
